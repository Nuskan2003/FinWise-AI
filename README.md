# 💰 FinWise AI

An AI-powered personal finance tracker that helps users manage their income, expenses, budgets, and savings while providing intelligent financial insights through AI.

---

## ✨ Features

- Secure user authentication
- Income and expense tracking
- Budget management
- Savings goals
- Interactive financial dashboard
- AI-powered spending insights
- Receipt scanning with OCR
- Monthly financial reports
- Responsive design
- Dark mode support

---

## 🛠️ Tech Stack

### Frontend

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

### Backend

- Next.js Route Handlers
- Prisma ORM

### Database

- PostgreSQL

### Authentication

- Better Auth

### AI & OCR

- OpenAI / Gemini API
- Tesseract.js

### Deployment

- Vercel
- Neon PostgreSQL
